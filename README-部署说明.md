# LindaOk GitHub Pages 分页最终版

本地预览：
```powershell
cd 你解压后的目录
python -m http.server 8020
```

访问：
- http://localhost:8020
- 首页第2页：/page/2/
- 归档第2页：/archives/page/2/
- 分类第2页：/categories/essay/page/2/
- 标签第2页：/tags/hexo/page/2/

部署到 GitHub Pages：
1. 新建仓库
2. 上传本压缩包解压后的所有文件到仓库根目录
3. Settings -> Pages -> Deploy from a branch
4. 选择 main / (root)


本版已合并：分页、黑白主题切换、白天搜索修复、关于页升级、留言页文案更新、页脚超链接。
